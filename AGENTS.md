# AGENTS.md

Este proyecto usa **`CLAUDE.md`** como fuente canónica de instrucciones para agentes de IA.

`AGENTS.md` existe como estándar abierto multi-herramienta (Cursor, Copilot, etc.). Para evitar
desincronización, tratá `CLAUDE.md` como la verdad y mantené este archivo como puntero.

👉 **Leé [`CLAUDE.md`](./CLAUDE.md)** para todo el contexto del proyecto: stack, reglas,
convenciones, comandos y flujo de trabajo.

> Tip: en macOS/Linux podés reemplazar este archivo por un symlink para que ambos sean idénticos:
> `ln -sf CLAUDE.md AGENTS.md`
